# 🏥 Medibot — Your AI Health Assistant

> "Understanding your symptoms, one search at a time."

Medibot is an intelligent symptom checker web application built with Python and Streamlit. 
Designed to help users understand their symptoms, possible causes, home remedies, 
and when to seek medical attention — all in one place.

## ✨ Features

- 🔍 **15+ Symptoms Covered** — fever, headache, cough, cold, stomach pain, back pain, chest pain, fatigue, dizziness, vomiting, diarrhea, sore throat, skin rash, eye irritation, joint pain and more
- 💊 **Home Remedies** — practical remedies for each symptom
- 🔬 **Possible Causes** — understand what might be causing your symptom
- 🚨 **Doctor Alert** — know exactly when to seek medical help
- 📱 **Multiple Symptoms** — type multiple symptoms at once separated by commas
- 🎨 **Beautiful UI** — dark theme with teal/purple gradient design
- ⚡ **Instant Results** — no loading, no waiting

## 🚀 Run Locally

**Step 1 — Clone the repository**
```bash
git clone https://github.com/anjanapriyavp-create/Medibot.git
cd Medibot
```

**Step 2 — Install dependencies**
```bash
pip install streamlit
```

**Step 3 — Run the app**
```bash
streamlit run app.py
```

**Step 4 — Open browser**

http://localhost:8501/

## 🎯 How to Use

1. Open the live app link
2. Type a symptom in the search box (e.g. fever, headache)
3. You can type multiple symptoms separated by commas!
4. Click **"Analyse Symptoms"**
5. View description, causes, remedies and doctor advice
